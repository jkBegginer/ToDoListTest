# 🗂️ ToDoList

> 할 일을 스마트하게 관리하는 웹 애플리케이션

## ✨ 기능

- ✅ 할 일 추가 / 수정 / 삭제
- 🖱️ 드래그 앤 드롭으로 순서 변경
- 🔖 카테고리 직접 추가/삭제
- ⭐ 우선순위 설정 (높음/보통/낮음)
- 📅 마감일 설정 + D-Day 표시
- 🔔 브라우저 알림 (마감일 임박 시)
- 📊 엑셀(CSV) 내보내기
- 🔁 반복 할 일 (매일/매주/매월)
- 🔍 실시간 검색 + 필터/정렬
- 🌙 다크/라이트 테마 전환
- 📱 반응형 디자인

## 🚀 사용법

`index.html` 파일을 브라우저에서 열면 바로 사용 가능합니다.  
별도 설치나 서버 없이 동작합니다.

## 🛠️ 기술 스택

- HTML5
- CSS3 (글래스모피즘 다크테마)
- Vanilla JavaScript
- localStorage (데이터 저장)

## 📁 파일 구조

```
ToDoList/
├── index.html    # 메인 HTML
├── style.css     # 스타일
├── app.js        # 앱 로직
└── README.md     # 설명서
```
